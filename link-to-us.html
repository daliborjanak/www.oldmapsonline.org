Similar maps widget

The OldMapsOnline provides widget for partners for a quick access to OldMapsOnline page. The "similar maps" widget is simple code embeddable to the websites which will create pretty button to right side of page.

Technical details

Only what you need is to add following code into web page with right identificators of map:

OldMapsOnline 
<a href="http://www.oldmapsonline.org/link/map/cuni/1095907" id="omo_link">OldMapsOnline</a>
<script src="http://www.oldmapsonline.org/similar/cuni/1095907.js" defer></script>

You can modify the style of widget with CSS. See example on jsfiddle.
